SociablePeak: A Social Media Web Platform

SociablePeak: A Social Media Web Platform for Personalized Connection and Content Through Graph Traversal and Filtering

  • Category: Laravel Web Project
  • Project date: June 2025

SociablePeak is a feature-rich social platform based on Laravel with three integrated user roles: Users can schedule posts, manage profiles, engage in encrypted one-on-one and group chats, view analytics, receive friend suggestions based on mutual connections, and get personalized ad recommendations. Admins oversee platform activity, handle user reports, manage accounts, and analyze user growth and engagement. Business users can create and publish ads, interact with users, and track ad performance through analytics. The platform includes features like two-factor authentication, real-time notifications, an AI-powered caption generator, and content-based filtering to enhance user experience and engagement.
